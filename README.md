
# Minishell

42Malaga Cursus team exercise.

Write a `C` Program that behaves like a shell.

## Requirements

Any linux based distro like a Mac computer or Linux system.

## Compile & Run

```
$ make
$ ./minishell
```

## Norminette

Check the Norminette norme errors using the Makefile by running:

```
$ make norme
```
