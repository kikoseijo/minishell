
# Minishell

42Malaga Cursus team exercise. Build a `C` Program that behaves as a shell.

## Requirements

Any linux based distro as Mac or Linux.

## Compile & Run

```
$ make
$ ./minishell
```

## Norminette

Check the Norminette norme errors using the Makefile by running:

```
$ make norme
```
